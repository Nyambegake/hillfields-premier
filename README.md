# Hillfields Premier Academy & Junior School Website

Hillfields Premier Academy & Junior School Website is a premium institutional web platform developed to support modern educational administration, learner engagement, and academic excellence in alignment with Kenya’s Competency-Based Curriculum (CBC) and Competency-Based Education (CBE) frameworks. Built specifically for Hillfields Premier Academy and Junior School in Kisii, Kenya, the system provides a professional digital presence while supporting the operational needs of parents, teachers, administrators, and learners.

The platform is designed using a modern full-stack architecture with Python Flask powering the backend, semantic HTML5 for structure, Tailwind CSS for premium styling, and Vanilla JavaScript for frontend interactivity. This ensures a fast, scalable, and maintainable application with clean separation between backend logic, UI design, and client-side interactions.

Key features include a responsive homepage showcasing Hillfields Academy’s academic vision, a CBC Compliance Hub that highlights curriculum implementation aligned with Kenya Institute of Curriculum Development standards, and a premium admissions portal that streamlines learner enrollment. The platform also includes a multi-role login system supporting Parent, Teacher, and Administrator access, enabling role-based navigation and functionality.

A core feature of the system is the protected administrative dashboard, which allows leadership profile management and dynamic content updates. The leadership team, including Daniel Hernandez (Deputy Principal & CBC Coordinator) and Kevin Nyambega Mongare (Head of Junior School Learning Areas), can be managed through a secure interface with real-time updates reflected on the live homepage.

The system emphasizes modern UI/UX standards with glassmorphism navigation, sophisticated color palettes, smooth micro-interactions, and competency-based assessment visualization. Interactive CBC competency cards enable users to explore the seven core competencies through assessment rubric modals, supporting transparent competency tracking and learner progress monitoring.

This project reflects Hillfields Academy’s commitment to innovation, academic excellence, and future-ready education in Kisii County.
